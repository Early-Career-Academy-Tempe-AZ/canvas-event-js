Atributos¶

El método attr modifica o adiciona atributos, su funcionamiento es similar al del método css de jQuery:

// Usando un objeto
ce.circle(140, 30, 20).attr({
    fill: "green",
    stroke: "black",
    scaleX: 2,
    lineWidth: 2,
});

// Único atributo
ce.rect(200, 100, 30, 50).attr("fill", "orangered");
