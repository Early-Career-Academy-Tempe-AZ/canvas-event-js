Eventos
=======

La forma de asociar eventos con objetos es igual a la usada por **jQuery**.
canvas-event soporta tanto eventos del :doc:`rat贸n <mouse-events>` como del :doc:`teclado <keyboard-events>`.

Todos los manejadores de eventos reciben dos argumentos: la instancia de canvas-event
a la que pertenece el objeto y el objeto evento.

.. js:function:: bind(name, callback)

    **name** es el nombre del evento (`mousedown, click, keydown...`) y ``callback`` una funci贸n
    a ejecutar cuando ocurra dicho evento.
    
.. code-block:: javascript

    ce.circle(50, 50, 30).draw();
    
    ce.bind("mousedown", function(c, e) {
        this.fill = "red";
    });

.. code-block:: javascript

        mousedown: function(c, e) {
            this.fill = "red";
        },
        
        mouseup: function(c, e) {
            this.fill = "black";
        }
    });
