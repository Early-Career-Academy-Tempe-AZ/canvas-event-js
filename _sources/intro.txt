Introducción
=============

El objetivo principal de canvas-event como su nombre lo indica, es el manejo
de eventos en el elemento canvas, ademas de reducir en gran medida la cantidad
de lineas necesarias para crear una aplicación.

.. code-block:: javascript

    // API nativa canvas 
    //===================
     
     var cv = document.getElementById("canvas"),
         ctx = cv.getContext("2d");
         
     // dibujar circulo rojo
     ctx.save();
     ctx.beginPath();
     ctx.fillStyle = "red";
     ctx.arc(50, 50, 30, 0, Math.PI*2, true);
     ctx.closePath();
     ctx.fill();
     ctx.restore();
     
      // Ejemplo canvas-event 
      //======================
      
      var ce = Cevent("canvas");
      
      ce.circle(50, 50, 30).attr("fill", "red").draw();
      
Muchas de las características de canvas-event como el encadenado de métodos y manejo de eventos
fueron "copiadas" de `jQuery <http://www.jquery.com>`_ lo cual facilita su aprendizaje
a personas familiarizadas con este framework.


Pasos previos
==============

canvas-event funciona en todos los navegadores modernos que tengan soporte para html5.
Los únicos requisitos son:

* Posicionar relativamente el elemento canvas.
* El canvas debe tener un id necesario para crear la instancia de canvas-event

.. code-block:: html

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8" />
        <title></title>
        <style>
            #canvas { position: relative; }
        </style>
    </head>
    <body>
    
    <canvas id="canvas" ></canvas>
    
    <script src="directorio/canvas-event.min.js"></script>
    </body>
    </html>
