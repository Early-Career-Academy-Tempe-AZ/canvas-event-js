Ejemplo zoom¶

Una de las nuevas características de canvas-event-js es la posibilidad de alejar y acercar (zoom) mientras los eventos del ráton siguen comportándose como se esperaría.

Use:

  • ctrl++ (acercar)
  • ctrl+- (alejar)
  • ctrl+0 (restaurar zoom)

Nota:* los objetos pueden ser arrastrados.

// La parte interesante
zoom.keydown("ctrl++", function(c, e){
    c.zoomIn();
    return false;
})
.keydown("ctrl+-", function(c, e){
    c.zoomOut();
    return false;
})
.keydown("ctrl+0", function(c, e){
    c.zoomTo(1);
    return false;
})
// hacer eventos del teclado globales
.setGlobalKeyEvents(true);
