Rutas SVG
=========

Para crear figuras mas complejas que simples :js:class:`rectángulos <Cevent.rect>` y 
:js:class:`círculos <Cevent.circle>` podemos hacer uso del objeto :js:class:`path <Cevent.path>` para dibujar
rutas SVG.

Una ruta SVG es un string que describe el camino a seguir para crear una figura.

.. code-block:: javascript

    ce.path("m50 50h30v-10h-10v-10h-10v10h-10v10").draw();
    
El ejemplo anterior dibuja un `tetrominó <http://es.wikipedia.org/wiki/Tetromin%C3%B3>`_ T
y se interpreta como:

* mover a coordenadas x=50, y=50
* linea horizontal de 30px
* linea vertical de -10px (hacia arriba)
* **...**

.. image:: _static/tetro.png

Para mas información puede referirse a la 
`documentación oficial del formato <http://www.w3.org/TR/SVG/paths.html#PathData>`_

Otra forma crear rutas es usando los métodos estáticos del objeto **Cevent**.

.. code-block:: javascript
    
    var cv = document.getElementById("canvas"),
        ctx = cv.getContext("2d");

    // Establecer contexto de dibujo
    Cevent.setContext(ctx)
    // crear ruta svg
    .m(50, 50).h(30).v(-10).h(-10).v(-10).h(-10).v(10).h(-10).z();
    
    ctx.fill();
    
A la hora de :doc:`extender <extend>` canvas-event estos métodos son de gran utilidad.
